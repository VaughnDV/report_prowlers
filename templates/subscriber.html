Report Prowlers

Please read before subscribing

Report Prowlers is an internet based communication service that allows a user to report information on suspected prowlers effortlessly, and conveniently, to their local Security Company or Neighborhood Watch

Subscribers (e.g: Security Companies) support the service financially and guide the continual efforts to improve the service

Upcoming Features

User registration via facebook, gmail or twitter
Online payment system
Restful API access
Mobile app for Android and Ios
Facial recognition

Pricing

The subscription price for EACH area is R150.00.
Payments are controlled by you the subscriber.

How the system works

Users register and Report Prowlers for free.
Subscribers receive the users reports via email.
The email will also note similar or recent reports for that area.
This is to aid the subscriber in deciding the level of response to any given report.
